# react-basic-animate

## ✨ Features

* useAnimate 动效。
* useAnimatePercent 动效百分比。
* useScreenScroll 滚屏。

## 📦 Install

```
npm i react-basic-animate --save
```

## 🔨 Usage

```
import { useAnimate } from 'react-basic-animate';
```

## 🖥 Development

```
$ git clone https://github.com/Alfred-sg/react-animate
$ npm install
$ npm run start
```

打开浏览器访问 http://127.0.0.1:8000。
